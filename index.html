Vampire Survivors Save Editor

Last updated: February 16th, 2022 for game version 0.2.12 — Source

Backup save location:

  1. Go to the game's installation directory (e.g. by right-clicking on it in Steam and selecting "Browse local files")
  2. From there, the backup save is at resources\app\.webpack\renderer\SaveDataBackup.sav

To load the backup save after editing it, open the game, go to the options menu, and click the "Restore backup" button twice.

NOTE: You need to press "Update checksum" after editing! Otherwise, the game will think the save is corrupted.
